# СберБанк — Демо-приложение

Полнофункциональный банковский сайт на Flask + SQLite.

## Быстрый запуск

```bash
# Установить зависимости
pip install flask flask-sqlalchemy flask-login flask-bcrypt

# Запустить
python app.py
```

Затем открыть: **http://localhost:5000**

## Функционал

- **Регистрация / Вход** — по email или телефону
- **Приветственный бонус** — 10 000 ₽ при регистрации (начисляется автоматически)
- **Несколько счетов** — текущий, сберегательный, кредитный (каждый с бонусом при открытии)
- **Переводы** — по номеру счёта между любыми пользователями
- **История операций** — с фильтрацией по счётам
- **Проверка получателя** — автоматическая проверка при вводе номера счёта

## Тестирование переводов

1. Зарегистрируйте **два аккаунта** (в разных вкладках / режим инкогнито)
2. Скопируйте номер счёта получателя из раздела "Счета"
3. В первом аккаунте зайдите в "Переводы", вставьте номер и отправьте деньги
4. Переключитесь на второй аккаунт — деньги уже на счету!

## Стек

- **Backend**: Python 3 + Flask + SQLAlchemy + Flask-Login + Flask-Bcrypt
- **Database**: SQLite (файл `sberbank.db`)
- **Frontend**: HTML5 + CSS3 + Vanilla JS

## Типы счетов и бонусы

| Тип счёта       | Бонус при открытии |
|-----------------|--------------------|
| Текущий         | 10 000 ₽           |
| Сберегательный  | 50 000 ₽           |
| Кредитный       | 100 000 ₽          |
